Не используй localhost, для ссылки на текущую vds, так как из docker контейнера localhost будет этот контейнер, а не vds на котором он развёрнут
Вместо этого используй полный ip адрес vds в сети.


Если docker compose показывает старую версию image то используй
docker compose build --no-cache
docker compose up


React делает client based routing поэтому ссылка http://localhost:38660/rooms/10 не работает адекватно.
Вместо этого нужно использовать http://localhost:38660/ и заходить в комнату вручную


Для дебага на локалке чтобы подключиться к копьютеру из контейнера используй тот адрес компьютера что отображается в твоём роутере
